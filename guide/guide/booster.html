小黑回国

史上最快，免费使用


国服加速

1. 下载并解压Netch，然后运行Netch.exe。

2. 点击Subcrible -> Manage Subscribe进入subsribe界面，在框中输入如下内容后点击Add/Modify保存:

   Remark: 任意

   Link: https://gzhpxb.com/node

3. 关闭subcribe界面，在主界面中点击Subscribe -> Update Servers From Subcribe Links以获取节点。

4. 点击Mode -> Create Process Mode:

   a. 在Remark中输入想要的名字

   b. 点击Scan，导入你想要加速的游戏所在的根目录。

   c. 点击Save，然后关闭界面。

5. 在主界面选择Server(推荐Amoy/Hong Kong)，并在Mode中选择你所创建的模式。

6. 点击右下角的Start即可加速。
